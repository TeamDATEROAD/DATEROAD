# DATEROAD Admin Dashboard

데이트로드 관리자 대시보드입니다.

## 기능

- 코스 관리
- 사용자 관리
- 통계 확인
- 신고 관리

## 개발 환경

- Node.js
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 환경 변수

`.env` 파일을 생성하고 다음 변수를 설정하세요:

```
NEXT_PUBLIC_API_URL=https://api.dateroad-main.p-e.kr
```

## 배포

Vercel을 통해 자동 배포됩니다.
